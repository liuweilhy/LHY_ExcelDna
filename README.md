# LHY_ExcelDna

#### 介绍
C#版本的ExcelAddIn插件，使用ExcelDna支持库。功能与本人的LHY_ExcelAddIn_VBA类似，但文件检索的功能有改进（支持Unicode文件名），快捷键未支持。
